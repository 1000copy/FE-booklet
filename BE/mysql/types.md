# mysql数据类型

　　数据类型是指列、存储过程参数、表达式和局部变量的数据特征，它决定了数据的存储格式，代表了不同的信息类型。mysql中的数据类型主要包括整型、浮点型、日期时间型和字符型。下面来详细介绍mysql中的数据类型

&nbsp;

### 整型

　　关键字UNSIGNED表示无符号值

　　数据库并没有布尔类型BOOLEAN，如果声明类型为BOOLEAN，则会被转换为TINYINT类型，true转换为1，false转换为0
